# svelte

Фронтенд был написан при помощи svelte и sveltekit, при этом использовалась библиотека для диаграмм ChartJS

## Установка

Для разворачивания svelte необходимо скачать установить nodeJS, зайти через bash в главный репозиторий и прописать следующую комманду

```bash
npm install
```

Эта комманда установит необходимые зависимости и инициализирует nodeJS

Следующая комманда запустит фронтенд сервер

```bash
npm run dev
```
